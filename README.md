# Hotel XVII – Boutique Hotel Landing

Static, premium landing page for Hotel XVII (Kadıköy). Built with plain HTML, CSS, and vanilla JS—no backend required.

## Structure
- `index.html` – markup and inline loader fail-safe.
- `style.css` – visual design, animations, responsive layout.
- `script.js` – interactive features (theme toggle, menus, modals, booking simulation, galleries).

## Run
Open `index.html` in a modern browser. No build step needed.

## Key Interactions
- Page loader with graceful timeout fail-safe.
- Mobile menu with focus trapping.
- Theme toggle (light/dark) with smooth transitions.
- Room detail modal with gallery and amenities.
- Booking demo flow with validation and faux availability modal.
- Gallery lightbox and scroll-reveal animations.

## Notes
- Frontend-only demo; no real reservation backend.
- Tested in modern evergreen browsers; prefers-reduced-motion is respected where applicable.
